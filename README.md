# linear-regression
Tools Used: Python, Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn, Jupyter Notebook. Key Skills: Data Cleaning, Data Visualization, Feature Engineering, Linear Regression, Model Evaluation, Machine Learning.
This project uses a Linear Regression model to predict the popularity of Spotify songs based on their audio features. The dataset contains various song characteristics such as danceability, energy, loudness, tempo, acousticness, speechiness, and valence. The goal is to understand how these features influence song popularity and build a predictive machine learning model.
Objectives
Perform data cleaning and preprocessing.
Explore relationships between audio features and song popularity.
Visualize data using charts and graphs.
Train a Linear Regression model.
Evaluate model performance using regression metrics.
Dataset
Dataset: Spotify Songs Dataset
Features: Danceability, Energy, Loudness, Tempo, Acousticness, Speechiness, Instrumentalness, Liveness, Valence, Duration, etc.
Target Variable: Popularity
Technologies Used
Python
Pandas
NumPy
Matplotlib
Seaborn
Scikit-learn
Jupyter Notebook
Project Workflow
Load and explore the dataset.
Clean missing and duplicate values.
Perform Exploratory Data Analysis (EDA).
Visualize feature distributions and correlations.
Split data into training and testing sets.
Train a Linear Regression model.
Evaluate model performance using:
Mean Absolute Error (MAE)
Mean Squared Error (MSE)
R² Score
Results
The Linear Regression model was trained to predict song popularity using Spotify audio features. The analysis helped identify the factors that have the greatest impact on song popularity and demonstrated the use of regression techniques in predictive analytics
